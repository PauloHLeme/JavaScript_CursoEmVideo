O que o JavaScript é capaz de fazer?

Cliente x Servidor

  1. Cliente
    2. A máquina que utiliza um serviço fornecido por um Servidor
  2. Servidor
    3. A máquina que armazena e envia os dados de serviços para o cliente
  3. Navegador
    4. Faz a ponte entre a chamada de informação do cliente e o recebimento dos arquivos entregues pelo servidor, para então abrir esse documento e carregar a informação solicitada
  • O Cliente pede a informação
  • O Servidor entrega uma cópia da informação
  • O Navegador lê e abre a informação solicitada

JavaScript é uma tecnologia muito mais voltada para ClientSide, FrontEnd, ou seja, utilizado para desenvolver a plataforma exibida para o cliente

O Desenvolvimento de uma aplicação FrontEnd utiliza, de forma primordial, 3 tecnologias

  1. HTML
    2. (conteúdo) Responsável pela apresentação do conteúdo, seja texto, imagens, áudio, vídeo
  2. CSS
    3. (estilo) Responsável pelo design, pelo estilo, pelo layout do site
  3. JavaScript
    4. (interação) Responsável pelas interações, as ações e programações
