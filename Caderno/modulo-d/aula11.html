Condições

Sequências

Todos os códigos que fizemos até hoje nas aulas foram sequenciais, ou seja, seguiam uma sequência lógica de acontecimentos, não podendo serem executados em fora de ordem e nem com etapas puladas.

Mas existem casos onde nós vamos querer executar um comando intermediário apenas se alguma condição for atendida.

Dessa forma, o código se bifurca em duas ou mais condições diferentes. Isso se chama Desvio Condicional

Condições em JavaScript

Para criarmos uma condição, vamos criar um bloco com o código if()

Estrutura


    if (condição){
        true
    }else{
        false
    }

Condição Simples

Condições simples contém apenas a instrução para a saída true, caso contrário, o código segue o fluxo natural.

Exemplo


    var vel
    vel = 60.5
    console.log(`A velocidade do seu carro é ${vel}Km/h`)
    if (vel >60){
        console.log('Você ultrapassou a velocidade permitida.
        Receba essa multa de presente')
    }
    console.log('Dirija sempre usando cinto de segurança')

No exemplo acima, a mensagem "você ultrapassou a velocidade permitida. Receba essa multa de presente." só será exibida se a condição vel >60 for atendida. Caso contrário, o programa pula o comando dentro do bloco da condição e segue funcionando normalmente a partir da próxima linha

Condição Composta

Quando a condição tem uma execução para true e outra para false.

Exemplo


    var país
    país = 'EUA'
    console.log(`Naturalidade: ${país}`)
    if (país.toLowerCase() == 'brasil') {
        console.log('Você é Brasileiro!')
    } else {
        console.log('Você é Estrangeiro')
    }

Nesse caso, temos a condição if e else. Se a condição for atendida, significa que o que está no primeiro bloco de código será executado, e o que estiver no segundo será ignorado. Caso o resultado da condição seja falso, o contrário acontece, sendo ignorado o primeiro bloco que está sob a condição if e executando o segundo bloco, que pertence ao else.
