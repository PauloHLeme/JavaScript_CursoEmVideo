Funções

O que são Funções?

No nosso dia a dia, temos diversas rotinas, tarefas que precisamos fazer. Algumas diariamente, outras ocasionalmente, mas todas tem uma certa rotina. Um conjunto de tarefas, atividades que precisamos fazer para executar essa função.

Algumas tarefas do dia a dia são automáticas, fazemos sem que alguém nos peça, fazem parte do nosso sistema diário. Outras acontecem porque alguém pediu. Sua mãe, seu marido, etc.

Pois bem, as Funções em JavaScript são exatamente a mesma coisa! Definindo bem, temos que:

  • São ações executadas assim que são chamadas ou em decorrência de algum evento.
  • Uma função pode recebver parâmetros e retornar um resultado.

Exemplo


    function parimp (n){        //n é o parâmetro
        if (n%2 == 0){          //essa linha é a ação que a
            return 'par'          função executa
        }else{
            return 'ímpar'      //ímpar é o retorno dado
        }
    }
    let res = parimp(11)        //parimp é a chamada que faz
                                  a função ser executada

Parâmetros Opcionais

Se definirmos um parâmetro formal dentro da função, mas não passarmos um valor para tal na chamada, aquele parâmetro vai receber como valor undefined. Para que isso não aconteça, podemos fazer uso dos Parâmetros Opcionais.

Ao definirmos o parâmetro formal e nomeá-lo, podemos definir um valor base para ele. Dessa forma, se nenhum valor for dado durante a chamada, ele terá o seu padrão pré-estabelecido.

Exemplo


    function soma(n1=0,n2=0) {
        return n1 + n2
    }
    console.log(soma(7))

Perceba que, diferente do exemplo anterior, nesse os parâmetros da function receberam um valor n1=0 e n2=0. Ao informarmos apenas o valor 7 na chamada, o n2 automaticamente assumiu o padrão 0, possibilitando o cálculo.

Recursos Extras

Funções possuem uma caixa de ferramenta muito grande! Isso que aprendemos nesse curso básico, é só o começo!

Para citar dois exemplos básicos:

  • Funções podem ser criadas dentro de variáveis
  • Funções podem ser chamadas dentro delas mesmas para se executarem
