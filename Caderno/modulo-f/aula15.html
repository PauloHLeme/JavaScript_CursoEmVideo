Variáveis Compostas

Variável simples vs composta

Variáveis simples só guardam um valor por vez. Cada vez que colocamos um valor, ela exclui o valor anterior.

Já as variáveis compostas, também conhecidas como arrays, são capazes de armazenar diversos valores em uma única variável, tornando nosso código ainda mais enxuto!

Dentro das variáveis compostas, teremos também classificações internas, para podermos identificar cada um dos valores armazenados nelas, e cada uma das variáveis receberá um índice identificador, numerado a partir de 0.

Estrutura

A variável composta é composta por:

  • Array: Ou em português, Vetor. A própria variável
  • Elemento: Onde ficam alocados o espaço da memória, o valor inserido nela e o índice identificador
  • Índice: o Índice, ou chave identificadora do valor específico dentro da variável
  • Conteúdo: O valor propriamente dito dentro do elemento da variável
"Um array é uma variável que tem vários elementos. Cada elemento é composto por seu valor e por uma chave de identificação." Guanabara, Gustavo - 2019

Exemplo


    let num = [5,8,4]
    
    //Aqui definimos uma variável de nome num,
    com os valores 5, 8 e 4 armazenados nos
    espaços 0, 1 e 2, respectivamente

Não esqueça!

Na linguagem de programação JavaScript, as ordens numeradas iniciam em 0, e não em 1.

Trabalhando os valores

Podemos também adicionar valores novos dentro de um array já existente. E isso é feito de uma forma bem simples! Basta declararmos onde essa variável vai se encaixar.

Usando o exemplo anterior, podemos querer adicionar um novo elemento logo em seguida do 4, então declaramos que num[3] = 6 e pronto, temos o novo valor 6 inserido no elemento 3.

Podemos ainda utilizar um método de nome num.push(7). Dessa forma, é inserido o valor 7 dentro de um novo elemento, que terá o identificador gerado automaticamente no último espaço do array.

Podemos ainda descobrir quantos elementos temos no nosso vetor, e pra isso vamos utilizar o parâmetro num.length. Ainda nos apoiando no nosso exemplo, se pedirmos o length do array num feito acima, neste momento ele valeria 5. Pois temos os 3 valores declarados no início, junto com a declaração do array, o quarto valor foi adicionado uma declaração adicional, e o quinto valor foi adicionado com o método push!

Outro método muito importante também é o num.sort(), ele é o nosso organizador, vai colocar nosso conteúdo, nossos valores, em ordem crescente dentro dos elementos. Facilitando a localização.

Exemplo


    let num = [5,8,2,9,3]

    num.sort()
    num.push(1)
    num[6] = 0
    console.log(`Nosso vetor é ${num}`)
    console.log(`O vetor tem ${num.length} posições`)
    console.log(`o primeiro valor do vetor é ${num[0]}`)

Se rodarmos esse exemplo, a saída que teremos vai ser:


    Nosso vetor é 2,3,5,8,9,1,0
    O vetor tem 7 posições
    o primeiro valor do vetor é 2

Exibindo valores

O exemplo que mostramos acima tem apenas 7 posições, e mesmo assim, se quiséssemos exibir um a um, precisaríamos escrever 7 comando console.log(num[]). Mas, como cada posição recebe um valor crescente, iniciando de 0, e em cada linha a única coisa que muda é o valor do código, podemos utilizar nossas velhas amigas, as Estruturas de Repetição!

Continuando com o exemplo da página, vamos usar uma estrutura for() para representarmos todos os seus valores de forma bem simples.

Exemplo


    for(let pos = 0; pos < num.length; pos){
        console.log (num[pos])
    }

Dessa forma, vamos ter tantos valores na variável pos quanto valores na variável num, gerando exatamente as repetições necessárias para utilizar o código com todos os dados do array!


    2
    3
    5
    8
    9
    1
    0

Comando "for in"

Existe ainda um comando em JavaScript que é feito especialmente para os nossos Arrays, que simplifica ainda mais a forma de utilização de todas as variáveis. O comando for in pode ser feito da seguinte maneira


    for(let pos in num){
        console.log(num[pos])
    }

Isso é a mesma coisa que dizer que, "para(for) cada posição(pos) em(in) num". E isso retornaria para nós o mesmo resultado que o comando for tradicional!

Procurando Valores

E se quisermos procurar um valor dentro de um array gigante? Não dá pra ficar olhanto todos os valores e caçando!

pra isso, temos o método num.indexOf(). Ele vai procurar no nosso Array qual a chave que contém um determinado valor. No nosso exemplo de var num = [2,3,5,8,9,0], se pedirmos num.indexOf(8), ele vai nos retornar o resultado 3, que seria a posição deste valor dentro da array. Da mesma forma, se pedirmos um valor que não existe na nossa variável, ele vai retornar como resultado -1, indicando que o valor desejado é inexistente.
