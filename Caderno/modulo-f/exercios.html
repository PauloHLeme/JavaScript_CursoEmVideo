Exercícios

01.Analisador de Números

  • Número (entre 1 e 100): O usuário deve inserir um número entre 1 e 100
  • Botão Adicionar: Botão para adicionar o item na lista de análise
  • Lista de análise: Uma lista mostrando todos os números que o usuário adicionou no sistema
  • Botão Finalizar: Vai exibir os dados analisados
  • Retornar Resultados:

    • Ao todo, temos x números cadastrados.
    • O maior valor informado foi n.
    • O menor valor informado foi y.
    • Somando todos os valores, temos z.
    • A média dos valores digitados é m.

Detalhes extras

Ao adicionar um valor, aparecer na lista "Valor n adicionado"

Se o usuário adicionar um valor inválido ou que já existe, retornar mensagem dizendo: "Valor inválido ou já encontrado na lista"

Se deixar caixa vazia, retornar: "Por favor, insira um valor"

Se o usuário adicionar mais um valor após o resultado já ter sido exibido, apagar o resultado anterior.

Se tentar finalizar sem nenhum valor na lista, retornar mensagem "Adicione valores antes de finalizar"
