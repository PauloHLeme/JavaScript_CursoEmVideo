Introdução a Objetos

O Array no JavaScript é heterogêneo, ou seja, aceita valores de diferentes tipos em cada um de seus elementos. Mas, como já vimos, os elementos são nomeados com números, e as vezes isso torna complicado o uso para aplicações mais avançadas.

Nesse caso, os objetos tornam o nosso serviço bem mais fácil, pois permitem também, entre outras funcionalidades, a nomeação de cada elemento e também a alocação de funções nos espaços reservados para valores!

Exemplo


    let amigo = {
        nome:'José',
        sexo: 'M',
        peso:85.4,
        engordar(p){
            console.log('Engordou')
            this.peso+= p
        }
    }
    
    amigo.engordar(2)
    console.log(`${amigo.nome} pesa ${amigo.peso}Kg`)
