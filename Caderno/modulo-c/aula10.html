Eventos DOM

O que são Eventos?

Eventos são ações que podem ser executadas com o elemento, dependendo de determinado gatilho que o dispare, segundo aquilo que configurarmos.

O que são Funções?

Funções são conjuntos de códigos que serão executados quando o Evento ocorrer

Cada conjunto de código é chamado de bloco. Os blocos são delimitados entre chaves {}.

Nomeamos os blocos através da funcion, e colocamos o nome normalmente de acordo com a ação que vai acontecer, podendo ou não incluir parâmetros.

Inserindo Eventos em um elemento

Podemos colocar a chamada para o evento de duas maneiras: No código HTML, dentro da tag, ou no código do script

No HTML

Para chamarmos o evento no código HTML, vamos dentro da tag desejada e chamaremos o método desejado, junto do prefixo "on"

Exemplo

Queremos criar uma interação com a nossa div quando clicarmos com o mouse nela. Para isso, faremos o seguinte:


    <div id="area" onclick="clicar()">
        Interaja...
    </div>
    
    //Nesse caso, o método é "click", então aplicamos o
    prefixo "on" adicionando o "click" logo em seguida.
    o "Clicar" é o nome da function que é declarada mais
    abaixo, dentro do script

    <script>
        var area = window.document.querySelector('div#area')
        function clicar(){
            area.innerText = 'Clicou!'
            area.style.background = 'red'
        }
    </script>

No Script

Para deixarmos o nosso HTML mais limpo, podemos criar listeners, que são elementos que analizam os disparos de eventos. Ao fazermos isso, utilizamos addEventListener() dentro do nosso script, da seguinte forma:

Exemplo


    <div id="area">
        Interaja...
    </div>

    <script>
        var area = window.document.querySelector('div#area')
        area.addEventListener('click' clicar)
        
        //Aqui adicionamos o EventListener para a variável "area",
        com o método "click". Note que nesse caso não utilizamos o
        prefixo "on", que é necessário apenas dentro da tag HTML.

        function clicar(){
            area.innerText = 'Clicou!'
            area.style.background = 'red'
        }
    </script>

Dessa forma, a nossa chamada está no JavaScript, deixando o foco do HTML para o que ele é feito, o conteúdo!
