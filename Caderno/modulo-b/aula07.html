Operadores (parte 1)

O JavaScript trabalha com uma vasta gama de operadores. Neste curso, vamos nos focar nos seguintes:

  • aritméticos
  • atribuição
  • relacionais
  • lógicos
  • ternário

Nessa primeira aula, vamos conhecer os Operadores Aritméticos e de Atribuição

Aritméticos

Os operadores aritméticos são os operadores matemáticos, utilizado para fazer cálculos


    5 + 2 == 7       // + é operador de soma
    5 - 2 == 3       // - é operador de subtração
    5 * 2 == 10      // * é operador de multiplicação
    5 / 2 == 2,5     // / é operador de divisão
    5 % 2 == 1       // % é operador de resto de divisão
    5 ** 2 ==  25    // ** é operador de potência

Não esqueça! - Ordem de precedência

Da mesma forma que na matemática existe hierarquia nos cálculos realizados (primeiro potência e raiz, depois multiplicação e divisão e por último adição e subtração), também acontece com os Operadores Aritméticos

  1. Parênteses ()
  2. Potências **
  3. Multiplicação, divisão e resto de divisão * / %
  4. Soma e subtração + -

Auto-atribuições

As variáveis também podem gerar atribuições para elas mesmas, utilizando fórmulas lógicas, como por exemplo:


    var n = 3       //atribuimos o valor 3 para n
    n = n + 4       //como n era igual a 3, agora passa a valer 7
    n = n - 5       //2
    n = n * 4       //8
    n = n / 2       //4
    n = n ** 2      //16
    n = n % 5       //ao final de todas as operações, n vale 1

Podemos ainda simplificar o código, pois se a variável está recebendo uma atribuição dela mesmo, podemos fazer o seguinte:


    var n = 3       //atribuimos o valor 3 para n
    n = n + 4       simplificando       n += 4
    n = n - 5       simplificando       n -= 5
    n = n * 4       simplificando       n *= 4
    n = n / 2       simplificando       n /= 2
    n = n ** 2      simplificando       n **= 2
    n = n % 5       simplificando       n %= 5

A grande maioria das linguagens atuais aceitam essa forma de simplificação! Olha que aprendizado valioso!

Incremento

Podemos também querer fazer incrementos consecutivos de um valor, e para isso temos o Operador de Incremento

exemplo


    var x = 5       //Atribuímos o valor de x como 5
    x = x + 1       //Incrementamos mais 1 a x, fazendo ele valer 6
    x = x + 1       //com o novo Incremento, agora ele vale 7
    x = x - 1       //com a subtração, temos ele valendo novamente 6

E também podemos simplificar o código, usando x++ ou x--, simples assim!
