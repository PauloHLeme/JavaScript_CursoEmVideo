Variáveis e Tipos Primitivos

Anteriormente, aprendemos os comandos window.alert(), window.confirm() e window.prompt(), mas não éramos capazes de armazenar os dados de entrada nas caixas de confirmação e de prompt.

Nessa aula, vamos utilizar as VARIÁVEIS parra armazenar os dados e utilizá-los!

Comentários

Os comentários servem para, como o próprio nome sugere, comentarmos o texto dentro do nosso código.

Em JavaScript, utilizamos os sinais de // ou /* e */. O primeiro serve para comentarmos em uma única linha, e o segundo para várias linhas de explicação

Exemplo

            
    window.alert('Minha primeira mensagem!') //exibe um alerta

    /*Abaixo, o confirm gera uma caixa de confirmação.
    Já o segundo, uma caixa para entrada de dados!*/

    window.confirm('Está gostando de JavaScript?')
    window.prompt('Qual é o seu nome?')

Variáveis

As Variáveis são "espaços" onde alocamos informações, dados. Cada variável consegue alocar apenas um valor por vez. Para receber um novo valor, ele deve eliminar antes o dado anterior.

Para criar uma variavel, utilizamos o código var, damos um nome a ela e então atribuir um valor a eles!

As variáveis também possuem tamanhos diferentes, para receber informações diferentes. Como por exemplo um tipo de var para Números, um para palavras, etc...

Identificadores

Identificadores são os nomes dados para uma variável, e eles seguem algumas regras específicas:

  • Podem começar com letra, $, ou _
  • NÃO podem começar com Números
  • É possível utilizar letras ou Números
  • É possível usar acentos e símbolos
  • NÃO podem conter espaços (deve ser substituido por _)
  • NÃO podem ser palavras reservadas (palavras já utilizadas pela linguagem, como por exemplo o próprio var.

Exemplo

    
    var veiculo1 = 'carro'
    
    Neste caso,

    var é a chamada para criar a variável
    veiculo1 é o Identificador, ou seja, o nome da variável, e
    carro é o valor dentro da variável, o dado que ela armazena

    Sendo assim, temos que:


    var veiculo1 = 'carro'

    é equivalente a dizer que a Variável var de Identificador veiculo1
    recebe o valor 'carro'!

Dicas

  • Maiúsculas e Minúsculas são diferenciáveis
  • Utilize nomes Coerentes para a variável, de acordo com o valor que utiliza, e não apenas n1, n2, n3, n4. Isso vai deixar o código confuso!

Tipos Primitivos

As variáveis podem receber inúmeros tipos de dados. Podemos guardar números, valores, nomes, sentenças, informações verdadeiras ou falsas, etc.

Para identificar ao programa como tratar determinado tipo de dado com mais facilidade, existem justamente os "Tipos de Dados". Os principais tipos, são chamados de Tipos Primitivos.

São eles, a princípio:

  • Number

    • São os dados representando valores. O JavaScript não diferencia o tipo de número, e trata todos eles sob esse mesmo tipo primitivo.

    Exemplos: 5; 10; -10; 3.5

    Mas ainda podem ser subdivididos em:

    • Infinity
    • NaN (Not a Number)
  • String

    • São palavras, sentenças, textos. Valores não numéricos e não binários, como por exemplo:

    "Curso em Vídeo"; 'Paulo'; `Avenida 10, número 199, Anhembi, São Paulo - SP`.

    Note que ao escrevermos o nome de uma variável do tipo string, temos que colocar ele dentro de "", '', ou ``. O uso específico de cada uma dessas formas de se escrever será tratado em uma aula futura.

  • Boolean

    • Valores binários interpretados como True ou False, ou seja, Verdadeiro ou Falso.

  • Null
  • Undefined
  • Object
  • Array
  • Function
