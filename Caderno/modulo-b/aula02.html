Tratamento de Dados

OBS:Nesta aula, vamos tratar mais especificamente sobre as variáveis do tipo number e string

Ao utilizar uma variável e guardar os dados, estamos com eles armazenados, mas ainda não fizemos nada com o essas informações. Podemos por exemplo pedir o Nome, Sobrenome, endereço e telefone de uma pessoa e exibir na tela por diversos motivos.

Para isso, não basta sabermos exibir o prompt pedindo a informação e criar a variável para armazená-la. Devemos saber Tratar os Dados.

Vamos usar como exemplo um site que peça, logo que você entra nele, o seu nome, para que ele possa saber como se referir ao falar com você!

Neste caso, temos que pedir o nome, através de um window.prompt, e também temos que criar uma variável que armazene essa informação. Da seguinte forma:

                
    var = 'nome'
    nome = window.prompt('Qual o seu nome?')

Em seguida, o site vai criar uma caixa de mensagem dando as boas vindas ao usuário, utilizando o nome dele. Nessa parte, temos um novo desafio, apresentar o texto padrão e acrescentar o nome do usuário, que será diferente para cada pessoa que acessar, mas que sempre estará armazenado em um mesmo lugar, na variável nome

.

Para isso, faremos da seguinte forma:

                
    var = 'nome'
    nome = window.prompt('Qual o seu nome')
    window.alert('É um grande prazer te conhecer, '
    + nome)

Observem que nesse caso, tudo o que está dentro de '' é o texto padrão, estático, exibido pelo alerta, seguido pelo sinal de +, que em JavaScript é uma Concatenação, e por fim, fora das aspas, digitamos o nome da variável, nesse caso, nome.

Dessa forma, se o usuário digitou seu nome Paulo, a caixa de alerta retornará a mensagem "É um grande prazer te conhecer, Paulo"

Transformar String em Número

Quando temos uma entrada de um prompt, recebemos a variável como uma string, mas há casos onde precisamos de números, para realizar cálculos. E agora? Como fazemos?

Para isso, existem os conversores de variáveis, como por exemplo o Number.parseInt(n) e o Number.parseFloat(n), que transformam um número que entrou como string em um número inteiro ou real, respectivamente.

Exemplo

                        
    var n1 = Number.parseInt(window.prompt('Digite 
    um número:'))
    var n2 = Number.parseInt(window.prompt('Digite 
    outro número:'))
    var soma = n1 + n2
    window.alert('A soma dos valores é: ' + soma)

Podemos também utilizar apenas a string Number. Nesse caso, o código irá definir se o valor é inteiro ou real.

Transformar Número em String

Funciona como a situação anterior, mas com o código String(n) ou n.toString().

Formatando Strings

Em versões anteriores tínhamos que ficar fazendo concatenações em série para declarar várias variáveis dentro de uma simples frase.

Nas versões atuais, ao utilizarmos a crase para escrever o texto que será exibido, podemos fazer o que é chamado de template string, que utiliza placeholders ${n} para exibir a variável dentro do texto, substituindo o uso de várias concatenações.

Concatenando

                        
    var n1 = Number(window.prompt('Digite um número:'))
    var n2 = Number(window.prompt('Digite outro número:'))
    var soma = n1 + n2
    window.alert('A soma entre ' + n1 ' e ' + n2 
    ' é igual a ' + soma)

Template String

                        
    var n1 = Number(window.prompt('Digite um número:'))
    var n2 = Number(window.prompt('Digite outro número:'))
    var soma = n1 + n2
    window.alert(`A soma entre ${n1} e ${n2} é igual 
    a ${soma}`)
