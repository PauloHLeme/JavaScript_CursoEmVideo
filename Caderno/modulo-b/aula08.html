Operadores (parte 2)

Nessa aula, vamos ver os operadores Relacionais, lógicos e ternários.

Operadores Relacionais

Os operadores relacionais os que estabelecem relações, como maior, menor, igual, etc...

Ao utilizarmos Operadores Relacionais, recebemos em retonrno um resultado Booleano, ou seja, True or False (verdadeiro ou falso)


    5 > 2   True        //maior que
    7 < 4   False       //menor que
    8 >= 8  True        //maior ou igual que
    9 <= 7  False       //menor ou igual que
    5 == 5  True        //Igual a
    4 != 4  False       //diferente de

Dica!

Se uma expressão contiver operadores Relacionais e Operadores Aritméticos, os Aritméticos tem precedência para serem executados!

Operadores de Identidade

Os Operadores Relacionais comparam valores, e não tipos de variáveis, portanto, se tivermos a var n1 = 5 e a var n2 = '5' e pedirmos para verificar se n1 == n2 o retorno será true, mesmo que um seja do tipo Number e outro do tipo String.

para verificarmos não só o valor, mas também o tipo, utilizamos os operadores de Identidade.

Exemplo


    5 === 5     true    //igual irrestrito
    5 === '5'   false
    5 !== '5'   true    //diferente irrestrito

Operadores Lógicos

Os operadores lógicos servem para, como o próprio nome diz, escrevermos expressões lógicas, e são eles:

  • ! Negação (NÃO lógico)
  • && Conjunção (E lógico)
  • || Disjunção (OU lógico)

Operações Lógicas


    Negação             Resultado
    !true               false
    !false              True


    Conjunção           Resultado
    true && true        True
    true && false       false
    false && true       false
    false && false      false


    Disjunção           Resultado
    true || true        True
    true || false       True
    false || true       True
    false || false      false

Operadores ternários

Os operadores ternários tem três partes, e utilizam os operadores ? e : juntos para determinar saídas para uma expressão, seja ela verdadeira ou falsa.

Definindo, temos:

           teste ? true : false

Exemplo


    var idade = 21

        //Define a idade como 21 anos

    idade >=18 ? 'MAIOR' : 'MENOR'

        //Testa se a variável idade é maior ou igual a 18
        anos. Caso "true", ocorre o que vem após o "?",
        caso "false", ocorreo que vem depois do ":"

        //Neste exemplo, o resultado seria "MAIOR"
